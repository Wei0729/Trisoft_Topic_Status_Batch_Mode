Unzip the Setup zip file

Double click the TopicStatusCommandModeSetup.msi

Change the file path in run.bat 

This tool takes three parameters. 
1. The first one is topic category. You can choose either "Topic", "Image" or "Both". 
2. The second parameter is the file path of input csv file.
3. The third parameter is the folder path of log file you want to save to.

The report will save into the same folder where the input csv file is.
